# Python Learning Practice

Welcome to your Python practice workspace! This repository contains practice questions to help you learn Python programming.

## Files

- `python_practice_questions.md` - Complete list of practice questions organized by difficulty
- `practice_template.py` - Template file where you can write your solutions

## Getting Started

1. Open `python_practice_questions.md` to see all the questions
2. Use `practice_template.py` or create new `.py` files to write your solutions
3. Start with Level 1 (Basics) and work your way up
4. Run your code using: `python your_file.py`

## How to Run Python Code

### On Windows (Command Prompt or PowerShell):
```bash
python practice_template.py
```

### Or if you have Python 3 specifically:
```bash
python3 practice_template.py
```

## Learning Path

1. **Level 1: Basics** - Variables, data types, simple operations
2. **Level 2: Control Flow** - If/else, loops, conditions
3. **Level 3: Lists and Strings** - Working with collections and text
4. **Level 4: Dictionaries and Sets** - Key-value pairs and unique collections
5. **Level 5: Functions** - Reusable code blocks
6. **Level 6: File Handling** - Reading and writing files
7. **Level 7: Object-Oriented Programming** - Classes and objects
8. **Level 8: Algorithms & Data Structures** - Problem-solving with Python

## Tips

- Don't rush! Take your time with each question
- Try to solve problems on your own first
- Experiment and make mistakes - that's how you learn
- Use print() to see what your code is doing
- Read error messages - they tell you what went wrong

Happy coding! 🐍
