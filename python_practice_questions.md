# Python Practice Questions

## Level 1: Basics

### Question 1: Hello World
Write a program that prints "Hello, World!" to the console.

### Question 2: Variables and Data Types
Create variables to store:
- Your name (string)
- Your age (integer)
- Your height in meters (float)
- Whether you like Python (boolean)
Print all variables with descriptive labels.

### Question 3: Simple Calculator
Write a program that takes two numbers as input and performs:
- Addition
- Subtraction
- Multiplication
- Division
- Modulus (remainder)
Print all results.

### Question 4: Area of a Circle
Calculate and print the area of a circle with radius 5. Use the formula: area = π × r²
(Use π = 3.14159)

### Question 5: Temperature Converter
Convert 98.6°F to Celsius using the formula: C = (F - 32) × 5/9

## Level 2: Control Flow

### Question 6: Even or Odd

- 90-100: A
- 80-89: B
- 70-79: C
- 60-69: D
- Below 60: F

### Question 8: Largest of Three
Write a program that takes three numbers and prints the largest one.

### Question 9: FizzBuzz
Print numbers from 1 to 100, but:
- For multiples of 3, print "Fizz"
- For multiples of 5, print "Buzz"
- For multiples of both 3 and 5, print "FizzBuzz"

### Question 10: Sum of Numbers
Calculate the sum of all numbers from 1 to 100 using a loop.

### Question 11: Factorial
Calculate the factorial of a number (n! = n × (n-1) × ... × 1)

### Question 12: Multiplication Table
Print the multiplication table for a given number (e.g., table of 5: 5×1=5, 5×2=10, ...)

### Question 13: Prime Number Checker
Check if a given number is prime (divisible only by 1 and itself).

### Question 14: Print Pattern
Print the following pattern:
```
*
**
***
****
*****
```
